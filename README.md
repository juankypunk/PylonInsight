# PylonInsight

**Open Battery Analytics for Pylontech Systems**

PylonInsight is an open-source platform for analyzing BatteryView exports from Pylontech battery systems.

The project focuses on:

- Preserving original BatteryView data.
- Building an open and documented data model.
- Importing historical campaigns into PostgreSQL.
- Detecting battery degradation using objective metrics.
- Comparing battery health across multiple inspection campaigns.
- Generating technical reports suitable for long-term monitoring and warranty support.

---

## Why PylonInsight?

BatteryView provides access to valuable diagnostic information, but it is primarily designed as a monitoring tool.

PylonInsight complements BatteryView by transforming exported data into a structured historical database, enabling advanced analytics, trend detection, and reproducible diagnostics.

## Project Goal

PylonInsight is an open-source platform for the historical analysis of Pylontech battery storage systems.

Its goal is to preserve the data generated by BatteryView, transform it into an open and well-documented data model, and provide objective tools to assess battery module health, detect anomalies, and monitor their evolution over time.

The project is designed to be transparent, reproducible, and extensible, ensuring that every conclusion can always be traced back to the original data.

## Design Principles

- Original BatteryView data is never modified.
- Every conclusion must be reproducible.
- All assumptions are explicitly documented.
- The project favors transparency over black-box algorithms.
- Historical data is more valuable than isolated measurements.
- The parser should support multiple BatteryView versions whenever possible.

## Philosophy

PylonInsight is not intended to replace BatteryView.

Instead, it preserves, organizes, and analyzes the information exported by BatteryView, providing objective tools to evaluate the health of Pylontech battery modules and detect long-term trends that are not visible in individual exports.

## Non Goals

PylonInsight is **not** intended to:

- communicate directly with the BMS;
- replace BatteryView;
- modify battery firmware;
- send commands to the battery system;
- bypass manufacturer protections.

Its purpose is exclusively to analyze exported diagnostic data.

**This project is in its early stages. The data model and file formats may evolve until version 1.0. Feedback, sample datasets, and contributions are very welcome.**
